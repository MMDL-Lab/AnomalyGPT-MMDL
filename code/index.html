工业异常检测

什么是工业异常检测？

工业异常检测是一种通过AI技术自动识别工业产品表面缺陷、结构异常的技术。系统会分析上传的图像，生成热力图标记出可能存在异常的区域。

如何上传图像进行检测？

您可以点击检测区域的上传区域，或直接拖拽图像到上传区域。系统支持常见的图像格式如JPG、PNG等，建议文件大小不超过5MB。

如何解读检测结果？

检测完成后，系统会显示原始图像和热力图。热力图中颜色较深的区域表示该位置存在异常的可能性较高，颜色越深代表置信度越高。

工业图像合成

图像生成功能有什么用？

图像生成功能可以根据您选择的参数生成各种工业场景的图像，可用于模型训练、数据增强或测试系统性能。生成的图像基于MVTec AD工业缺陷数据集。

如何选择生成参数？

模型类型：GAN，DIFFUSION，多模态可控异常生成模块。
图像类型：对应不同工业场景（机械零件、电子元件等）。
复杂度：值越高，生成的图像可能包含更多细节或异常特征。

记录管理

如何查看历史记录？

点击导航栏中的"会话历史"或"生成记录"可以查看相应的历史操作记录，包括图像预览和详细参数。记录会保存在浏览器本地存储中。

如何保存结果？

在每个结果区域都有"保存结果"按钮，点击即可将图像保存到本地。在记录列表中，每个记录项也有下载按钮可供使用。

常见问题

上传图像失败怎么办？

请检查图像格式是否正确（支持JPG、PNG），文件大小是否过大（建议小于5MB）。如果问题持续，请刷新页面重试或检查网络连接。

处理时间过长怎么办？

图像处理时间取决于图像复杂度和服务器负载，通常在几秒内完成。如果长时间无响应（超过30秒），请刷新页面重试。

数据集支持哪些类型？

系统基于MVTec AD数据集，支持金属零件、电子元件、纺织品等多个类别的工业图像，具体包括：metal_nut、screw、cable、transistor等。
